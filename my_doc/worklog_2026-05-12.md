## 実装済み機能の概要

### StackChanMind（`firmware/src/StackChanMind.h/.cpp`）
スタックちゃんの感情状態を一元管理するクラス。グローバルインスタンス `stackChanMind`。
- 感情種類: happy / neutral / sad / angry / doubt / sleepy
- `setEmotion(name)`: 感情更新 + 即座に表情反映
- `applyExpression()`: 発話後など、表情だけ再適用する

### AckSound（`firmware/src/AckSound.h/.cpp`）
STT 完了後に感情に応じた相槌 MP3 を SD カードから再生する。
- SD カードパス: `/ack/{speakerId}/{filename}.mp3`
- `SC_ExConfig.yaml` の `ack.speakerId` で話者 ID を指定（-1 で tts.voice を自動使用）

### ChatGPT の感情連動（`firmware/src/llm/ChatGPT/ChatGPT.cpp`）
- chat() の各ループで現在の感情をシステムロールに追記
- LLM に返答末尾への `[META]{"emotion":"happy"}[/META]` 付与を要求
- execChatGpt() で [META] タグを解析し `stackChanMind.setEmotion()` を呼び出してタグを除去

### PlayMP3 の拡張（`firmware/src/driver/PlayMP3.cpp/.h`）
- `playMP3SDWithExpression(filename, expr)` を追加
- 再生中は指定表情、再生後は `stackChanMind.applyExpression()` で元に戻す
